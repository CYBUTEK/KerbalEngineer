Description

This plugin will allow you to view important statistics about your ship and it's orbit during both flight, building and in the tracking station. These statistics help you plan and execute your missions with greater accuracy.

Installation

  • Copy the 'KerbalEngineer' folder into the 'GameData' folder located within your Kerbal Space Program installation directory.
  • The 'Parts' folder including the 'ER7500' and 'EngineerChip' is optional and are only required if running in module mode.
